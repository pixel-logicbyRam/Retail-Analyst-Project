# Retail Analyst Project

## Overview
This project analyzes retail sales data to identify trends, KPIs, and actionable business insights using Python.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Google Colab

## Key Objectives
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Sales and customer analysis
- Business insights & recommendations
